ApowerREC 支持三种核心录制方式：

- **全屏录制**：最常用，直接录整个显示器画面
- **区域录制**：自定义框选范围，只录你需要的那块区域

三种模式切换很快，主界面点两下就搞定，不需要翻设置菜单。
